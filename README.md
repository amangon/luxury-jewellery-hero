# 💎 Luxury Jewellery Store Frontend

A premium luxury jewellery e-commerce frontend application built with **React.js, JavaScript, CSS, and modern UI technologies**.

This frontend provides a beautiful shopping experience for customers with product browsing, product details, cart, wishlist, authentication, and integration with backend APIs.

# ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/amangon/luxury-jewellery-hero.git
```

Go inside project folder:

```bash
cd luxury-jewellery-hero
```

Install dependencies:

```bash
npm install
```

---

# 🔐 Environment Variables

Create a `.env` file in the root directory.

Add backend API URL:

```env
VITE_API_URL=https://your-backend-url.onrender.com/api
```

---

# ▶️ Run Project

Start development server:

```bash
npm run dev
```

Build project:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
